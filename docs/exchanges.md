## Cryptocurrency Exchanges

### Major Centralized Exchanges (CEX)

#### Binance
- **Website:** binance.com
- **Volume:** #1 globally
- **Spot fees:** 0.1% (maker), 0.1% (taker)
- **Futures:** 0.02% / 0.04%
- **KYC:** Required for full features
- **Countries:** Most (not US)

**Pros:**
- Highest liquidity
- Most trading pairs
- Low fees
- Advanced features

**Cons:**
- Regulatory scrutiny
- Complex interface
- Past security incidents

**Best for:** Advanced traders, altcoins

---

#### Bybit
- **Website:** bybit.com
- **Volume:** #2-3
- **Spot fees:** 0.1% / 0.1%
- **Futures:** 0.02% / 0.055%
- **KYC:** Optional (limited)
- **Countries:** Most

**Pros:**
- Great derivatives
- Good API
- Fast execution
- Competitive fees

**Cons:**
- Limited spot pairs vs Binance
- Fewer fiat options

**Best for:** Derivatives, copy trading

---

#### OKX
- **Website:** okx.com
- **Volume:** Top 5
- **Spot fees:** 0.1% / 0.1%
- **Futures:** 0.02% / 0.05%
- **KYC:** Optional
- **Countries:** Most

**Pros:**
- Good for Web3/DeFi
- P2P trading
- Competitive fees

**Cons:**
- Less known in West
- Complex interface

**Best for:** DeFi enthusiasts, P2P

---

#### Kraken
- **Website:** kraken.com
- **Volume:** Top 10
- **Spot fees:** 0.16% / 0.26% (tier 1)
- **Futures:** 0.02% / 0.05%
- **KYC:** Required
- **Countries:** US, EU, UK

**Pros:**
- Strong regulation
- Good security
- Fiat support
- Professional features

**Cons:**
- Higher fees
- Slower expansion
- Fewer altcoins

**Best for:** US users, regulated environment

---

#### Coinbase
- **Website:** coinbase.com
- **Volume:** Top US
- **Spot fees:** 0.0% - 0.6% (maker-taker)
- **KYC:** Required
- **Countries:** 100+

**Pros:**
- Easiest to use
- Strong regulation
- Good for beginners
- Insurance on assets

**Cons:**
- High fees
- Limited advanced features
- Higher spreads

**Best for:** Beginners, US users

---

### Decentralized Exchanges (DEX)

#### Uniswap
- **Type:** AMM (Automated Market Maker)
- **Network:** Ethereum, Arbitrum, Optimism
- **Fees:** 0.3% per swap
- **KYC:** None

**Pros:**
- No KYC
- Access to new tokens
- Non-custodial

**Cons:**
- Higher slippage
- Gas fees
- No limit orders

---

#### PancakeSwap
- **Type:** AMM
- **Network:** BNB Chain
- **Fees:** 0.25% per swap
- **KYC:** None

**Pros:**
- Very low fees
- Fast
- Many pairs
- Yield farming

**Cons:**
- Scam tokens
- Centralization concerns

## Stock Brokers

### US Brokers

#### Interactive Brokers (IBKR)
- **Fees:** $0 for US stocks
- **Min deposit:** $0
- **Countries:** 200+

**Pros:**
- Low fees
- Global access
- Advanced tools
- Great API

**Cons:**
- Complex interface
- High learning curve
- Inactivity fees

**Best for:** Active traders, international

---

#### Fidelity
- **Fees:** $0 for US stocks
- **Min deposit:** $0
- **Features:** Good research, retirement accounts

**Pros:**
- Excellent research
- No fees
- Good customer service

**Cons:**
- No fractional shares (in some accounts)
- Limited international

**Best for:** Long-term investors

---

#### Robinhood
- **Fees:** $0
- **Min deposit:** $0
- **Pros:** Simple, mobile-first
- **Cons:** Limited research, no ACH transfers

**Best for:** Beginners

---

### International Brokers

#### eToro

- **Min deposit:** $10-200

**Pros:**
- Copy trading
- Social features
- Multi-asset
- Easy interface

**Cons:**
- Higher spreads
- Withdrawal fees
- Limited order types

**Best for:** Copy trading, beginners

**Cons:**

- No US crypto

## Selection Criteria

### Security (Critical)
- Regulation status
- Cold storage policy
- Insurance coverage
- History of hacks
- Two-factor authentication

### Fees
- Trading fees
- Withdrawal fees
- Deposit fees
- Inactivity fees
- Spread costs

### Features
- API access
- Trading platforms
- Order types
- Leverage available
- Mobile app

### Accessibility
- Your country
- Fiat on/off ramps
- Payment methods
- Verification speed

### Support
- Language
- Response time
- Quality of help

## For Trading Bot Development

### API Quality

**Best for bots:**
1. **Binance** - Best API, webhooks, unified account
2. **Bybit** - Good documentation, fast
3. **Kraken** - Solid, regulated
4. **IBKR** - Best for stocks
5. **OANDA** - Best for forex

### Key API Features Needed
- REST API for orders
- WebSocket for real-time data
- Market data access
- Order management
- Position/PnL tracking
- Testnet availability

### Recommendation
Start with **Binance** (crypto) or **IBKR** (stocks) for bot development:
- Best documentation
- Comprehensive APIs
- Testnet available
- Large liquidity
